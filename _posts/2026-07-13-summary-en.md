## [DOOMQL: Doom-like game powered by SQLite SQL queries](https://simonwillison.net/2026/Jul/13/doomql/#atom-everything) ⭐️ 8.0/10

Peter Gostev released DOOMQL, a Doom-like game that uses SQLite as the entire game engine, controlling movement, enemies, and rendering via SQL queries. The game runs as a Python terminal script and includes a ray tracer implemented in SQL using recursive CTEs. DOOMQL is a highly creative demonstration that pushes the boundaries of what SQLite can do, repurposing a database engine for real-time game rendering and logic. It challenges conventional assumptions about database capabilities and showcases novel integration of SQL with interactive applications. The game stores its entire state in a SQLite database at /tmp/doomql/.doomql/doomql.sqlite, which can be explored using Datasette. The rendering includes a full ray tracer implemented in a single large SQL query with recursive CTEs.

**Background**: SQLite is a self-contained, serverless, zero-configuration SQL database engine that is widely embedded in applications. This project leverages SQLite's recursive common table expressions (CTEs) and virtual tables to handle game logic and real-time pixel rendering, demonstrating that databases can be used for tasks far beyond data storage.

## [CoT is a scaling trap; latent reasoning and BDH emerge](https://www.reddit.com/r/MachineLearning/comments/1uviru5/chain_of_thought_is_a_scaling_trap_the_next_wave/) ⭐️ 8.0/10

A Reddit post argues that Chain-of-Thought reasoning is a scaling trap due to faithfulness and cost issues, and advocates for latent reasoning approaches such as Coconut, RecursiveMAS, and BDH. The post also highlights the interpretability gap with latent reasoning and proposes a symbolic outer loop governance layer for auditability. This critique challenges the dominant paradigm of using Chain-of-Thought reasoning in language models, suggesting a shift towards latent reasoning that could reduce costs and improve faithfulness. The discussion also underscores the need for new governance frameworks to maintain accountability when models reason internally. Coconut enables breadth-first search in latent space, BDH achieves 97.4% top-1 accuracy on Sudoku Extreme without CoT, and RecursiveMAS uses latent embeddings for agent communication. The post identifies time recurrence as a challenge for latent reasoning in streaming contexts and suggests an outer loop with DAG-based planners and deterministic verification.

reddit · r/MachineLearning · /u/meowsterpieces · Jul 13, 17:50

**Background**: Chain-of-Thought (CoT) reasoning prompts models to generate step-by-step explanations, which can improve performance but also increases token usage and can produce unfaithful rationales. Latent reasoning methods aim to perform internal computations without generating intermediate text, potentially reducing cost and improving efficiency. However, this opacity raises concerns in high-stakes applications, leading to proposals for hybrid governance layers that combine symbolic planning with neural computation.
